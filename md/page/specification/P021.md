### <상세설명>  
오랫동안 편집되지 않은 문서들의 목록을 가져온다.

성공했을 경우 반환
{
    'success':true,
    [
        {
            number: ?,
            title: '...'
        }
    ]
}

실패했을 경우 반환
{
    'success':false,
    'error':'에러 내용'
}
